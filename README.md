# C# 1.0 Mini-Programs - Full Solution

This solution contains **24 small console projects** that demonstrate **C# Version 1.0 (2002)** features.
It targets **.NET 10.0** but enforces **C# 1.0 syntax** via `Directory.Build.props` (`<LangVersion>ISO-1</LangVersion>`).

We also disable the SDK's autogenerated TargetFramework attribute and include our own (without `global::`) to avoid newer syntax creeping in.

## Build & Run

```bash
dotnet --version         # should show 10.0.100-rc.1.25451.107 (or newer 10.x)
dotnet clean
dotnet restore
dotnet build
dotnet run --project P01_HelloWorld
```

If you see restore errors about `net10.0`, ensure your SDK is 10.x and there's no `global.json` pinning an older version outside this folder.

## What's intentionally not used
No generics, lambdas/anonymous methods, LINQ, `var`, `yield`, nullable value types, partial classes, expression-bodied members, string interpolation, etc.

# Guided README (P01-P24)

This repository demonstrates **C# 1.0 (2002)** features using 24 tiny console projects. It stays inside the C# 1.0 language surface (no generics, lambdas, LINQ, `var`, `yield`, etc.). The goal is to learn/teach the essentials with minimal scaffolding.

Move through the projects in order; each adds one concept.

---

## P01_HelloWorld - Hello, World
**Concepts:** Console I/O, `Main`.
```csharp
static void Main()
{
    Console.WriteLine("Hello, World!");
}
```

## P02_ClassesAndStructs - Classes and structs
**Concepts:** Reference vs value types, fields.
```csharp
public class PersonClass { public string Name; }
public struct PointStruct { public int X, Y; }
```

## P03_Interfaces - Interfaces
**Concepts:** Defining and implementing interfaces.
```csharp
interface IShape { void Draw(); }
class Circle : IShape { public void Draw(){ Console.WriteLine("Drawing a circle."); } }
```

## P04_Indexers - Indexers
**Concepts:** Indexer syntax with `this[int]`.
```csharp
public string this[int index]
{
    get { return days[index]; }
    set { days[index] = value; }
}
```

## P05_Arrays - Arrays
**Concepts:** Array creation and iteration.
```csharp
int[] numbers = new int[5] { 1, 2, 3, 4, 5 };
for (int i = 0; i < numbers.Length; i++) { Console.WriteLine(numbers[i]); }
```

## P07_Exceptions - Exceptions
**Concepts:** `try/catch/finally`, parsing input.
```csharp
try { int n = int.Parse(input); }
catch (FormatException) { Console.WriteLine("Error: Invalid number format."); }

```

## P13_Conversions - User-defined conversions
**Concepts:** `implicit`/`explicit` operator conversions.
```csharp
public static implicit operator Celsius(Fahrenheit f)
{
    return new Celsius((f.Degrees - 32) * 5 / 9);
}
public static explicit operator Kelvin(Fahrenheit f)
{
    return new Kelvin((f.Degrees - 32) * 5 / 9 + 273.15);
}
```

## P21_EqualsAndGetHashCode - Equality
**Concepts:** `Equals`/`GetHashCode` consistency.
```csharp
public override bool Equals(object obj)
{
    Point p = (Point)obj;
    return X == p.X && Y == p.Y;
}
public override int GetHashCode()
{
    return X.GetHashCode() ^ Y.GetHashCode();
}
```
